# Phone Number Tracker

A simple Python application to track and retrieve information about phone numbers using the phonenumbers library.

## Features

- Validate phone numbers
- Get carrier information
- Retrieve location details
- Timezone information

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd phone-number-track
   ```

2. Create a virtual environment:
   ```
   python -m venv .venv
   ```

3. Activate the virtual environment:
   - On Windows: `.venv\Scripts\activate`
   - On macOS/Linux: `source .venv/bin/activate`

4. Install dependencies:
   ```
   pip install phonenumbers
   ```

## Usage

Run the main script:
```
python main.py
```

Follow the prompts to enter a phone number and view the information.

## Requirements

- Python 3.6+
- phonenumbers library
